# Marc Lou “$0 → 1,000 visitors” Launch Stack — Adapted for FormBeep

Source (tweet): Marc Lou @marclou (Apr 1, 2025)
Captured: 2026-04-01

## The principle (what’s actually true)
This isn’t “one magic post.” It’s **multi-surface distribution**:
- same story → repackaged into 8–10 formats
- posted where your buyers/builders already hang out
- creates fast feedback + some traffic + some backlinks

Not guaranteed 1,000, but it’s one of the highest-leverage $0 playbooks when executed cleanly.

---

## FormBeep adaptation (non‑US / WhatsApp-heavy focus)

### Surfaces we SHOULD use now
1) Reddit: r/SideProject + 1–2 other subs (rules-checked; story-first)
2) Indie Hackers: story post (align with /blog/why-formbeep tone)
3) Directories (DR + discovery): Uneed, TinyLaunch, PeerPush, TrustMRR, etc. (2/day)
4) Twitter/X: 280-char “lesson” + 1 screenshot/GIF
5) Cold outreach: micro-influencers (Webflow/Framer/agency creators in MENA/India/EU)

### Surfaces we should DELAY / use only if angle is strong
- Product Hunt: do later when we have tighter conversion + 1–2 testimonials
- Show HN: only if we have a genuinely interesting technical hook (e.g., “form-to-WhatsApp without backend” with implementation details)

### Video (still worth doing)
- Short video (IG/Reels): 20–35s “problem → demo → result”
- Longer video (YouTube): 2–4 minutes “setup walkthrough + why this exists”

---

## Minimum viable “Launch Stack” asset kit (must exist before posting)
- 1 core narrative paragraph (why-formbeep style)
- 3 screenshots + 1 short GIF (or Loom)
- 1 demo flow: submit form → WhatsApp ping
- 1 call-to-action variant:
  - If link allowed: https://formbeep.com/blog/why-formbeep/ or /pricing
  - If link not allowed: ask for feedback; no link

---

## Execution cadence (ties into Distribution Block)
Daily (15 min):
- 2 directory submissions
- 3 community comments
- 1 outreach touch

Weekly “Launch Stack Burst” (45–60 min, once/week):
- 1 Reddit post (r/SideProject)
- 1 IndieHackers story or update
- 1 short video (reel)
- 1 tweet

---

## Success metric (7 days)
- 10+ directory approvals submitted
- 2 posts + 10–15 comments on Reddit
- 1 IndieHackers post
- 1 short video published
- 5 influencer outreach messages

Expected leading indicators:
- referral traffic from directories within 48–72h of approvals
- more DMs/questions (“how did you do WhatsApp alerts?”)
- GSC impressions increase (even before clicks)
