## Reply-guy mode (10 replies/day) — how to do it without wasting time

Daily block (20–30 min):
1) Pick 3 lanes (rotate): indie SaaS building, SEO/content, analytics/ops
2) Find 10 posts in-lane (mix big accounts + mid-tail builders)
3) Reply with: one edge-case + one mini playbook + one metric
4) Log outcomes (so you learn what converts)

Reply types that win:
- “Edge case that bit me”
- “If I had to do this today: 1/2/3”
- “Small disagree: this matters less than X because Y”

Hard rule:
- No links in the reply unless asked. If needed, link in a follow-up.
