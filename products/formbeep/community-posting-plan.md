# FormBeep Community Posting Plan

Last updated: 2026-03-30
Owner: Moxie
Status: active draft

## Goal
Create a repeatable motion for traffic and awareness from Reddit, X, and niche communities without spamming.

## Public X access status
- Public browsing works in headless browser for profiles/pages that are visible without login.
- No X API credentials currently configured.
- Therefore: can research public profiles/posts opportunistically, but cannot rely on authenticated posting/automation through official API yet.

## Community motion
### Reddit
Focus on:
- answering relevant threads
- sharing implementation advice
- posting useful walkthroughs only where appropriate
- looking for lead-response, contact form, WhatsApp, agency, WordPress, Webflow, and no-code discussions

### X
Use for:
- founder posts about use cases, integrations, and build-in-public proof
- replies to relevant conversations from agencies, freelancers, Webflow/WordPress builders, and lead-gen operators
- traffic support, not core acquisition yet

### Other communities
- Indie Hackers
- Webflow communities
- WordPress communities
- no-code groups
- freelancer/agency communities

## Timing principle
- Check for opportunities during the hourly heartbeat and heavier scans in background
- Notify Rishi with ready-to-post copy when a timely posting/commenting opportunity is found

## Posting principles
- teach first
- use actual examples
- show how to solve a real lead-response problem
- avoid repetitive self-promo
