## 3. Competitor Keyword Gaps

### Beepmate
- **Search presence:** Minimal. No significant HN discussions, GitHub repos, or indexed pages found for "Beepmate" in form-tech context
- **Gap opportunity:** Since Beepmate has weak search presence overall, FormBeep can easily outrank it on comparative and branded searches
- **Recommendation:** Create a "FormBeep vs Beepmate" comparison page immediately; low-hanging fruit

### Formspree (Category Leader)
| What Formspree ranks for | FormBeep gap |
|--------------------------|--------------|
| "form endpoint" | High — FormBeep doesn't position as form endpoint |
| "contact form hosting" | High — Not a current angle |
| "form submission service" | Medium — Can claim adjacent |
| "static site form handling" | Medium — Doc pages exist but lack ranking power |
| "form backend" | Low-mid — Differentiated but not claimed as keyword |

### FormBee (GitHub competitor — 762 stars, open-source)
| What FormBee has that FormBeep lacks |
|--------------------------------------|
| GitHub presence (huge in developer search) |
| Open-source angle ranks for "open source form backend" |
| Self-hosted alternative positioning |
| Data privacy / GDPR positioning |

**Gap Action:** FormBeep should target "hosted WhatsApp form notifications" to differentiate from FormBee's self-hosted/open-source positioning. Don't compete on "open source form backend" — compete on "instant alerts via WhatsApp."

### FormKeep & Formspree (General Form-as-a-Service)
| What they do well | What they miss |
|-------------------|----------------|
| General "form to email" / "form endpoint" SEO | Nobody targets WhatsApp as notification channel |
| High domain authority | Slow notification (email-based); FormBeep's WhatsApp is real-time |
| Established brand | Complex setup; FormBeep's value is "5-minute setup" |

**Key gap FormBeep owns exclusively:** Nobody else is targeting "form to WhatsApp" as a category. This is defensible whitespace.
