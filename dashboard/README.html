Rules

This dashboard is intended to be public. Only publish sanitized, aggregated data.

  • Data source: public_snapshot.json
  • Render: app.js reads snapshot and paints UI
  • UI: 7.css (scoped) + custom monochrome CSS

Never publish:

  • Emails / phone numbers
  • API keys / tokens
  • Chat IDs
  • Full URLs with query strings
  • Raw logs

OK to publish:

  • Weekly totals (PV, visitors, signups)
  • Counts of blockers
  • Role-only statuses (IDLE / IN_PROGRESS / BLOCKED)
  • Counts of active jobs

If you want, I can add a generator script later that writes a sanitized snapshot from internal state (private), while keeping this site static and safe.
